Программа модифицирует учетные записи пользователей в MS Active Directory, используя справочник предприятия в формате Excel. Добавляет атрибуты ученой записи: Адрес, кабинет, компания, департамент, должность, городской телефон, внутренний телефон. 
Вначале нужно импортировать список учетных записей из Active Directory в формате .cvs и загрузить его в программу. 
Затем программа сравнивает этот список с Excel-справочником и формирует 4-е области. 
1. Список учетных записей не найденных в Excel-справочнике. 
2. Список совпадений в Excel-справочнике, однофамильцев с одинаковым именем и отчеством.
3. Заполнения атрибутов для каждой учетной записи. Показывает ход выполнения.
4. И результат, который используется для модификации атрибутов учетных записей в MS Active Directory.
